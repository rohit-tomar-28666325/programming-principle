LincolnLibrary

1. Project Overview
   - Description: The LincolnLibrary project is a library management system that allows users to manage books, users, and transactions.
   - Modes: Console application, Server, Client

2. Dependencies
   - C++17 or later
   - GNU Make
   - g++ (GNU Compiler Collection)

3. Installation
   - Build the project using `make`
     make

4. Running the Code
   - Run the Application:
     ./bin/LincolnLibrary

5. Usage Instructions
   - When running the console application, you will be prompted with the following options:
     1. Run Console Application
     2. Run as Server
     3. Run as Client
